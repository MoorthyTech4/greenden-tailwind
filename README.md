# Greenden – Tailwind CSS Project

Greenden is a simple and responsive web project built using Tailwind CSS. This project includes a Home page, Products page, and Contact page, designed to demonstrate clean UI structure and modern styling using utility-first CSS.

The Home page provides an overview of the brand with a visually appealing layout. The Products page showcases items in a structured and responsive grid format. The Contact page allows users to connect through a simple and user-friendly form design.

This project focuses on improving front-end development skills by using Tailwind CSS for styling, ensuring responsiveness across different screen sizes, and maintaining clean and organized code.

Technologies used
HTML5
Tailwind CSS

Features
Responsive design for all devices
Modern UI using Tailwind utility classes
Multi-page navigation (Home, Products, Contact)
Clean and structured layout

Purpose
This project is created for learning and practicing Tailwind CSS, layout design, and building responsive web pages.
